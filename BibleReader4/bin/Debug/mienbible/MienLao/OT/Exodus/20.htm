ທິນ-ຮູ່ງ ປຸນ ຕສຽບ ຕິ້ວ ເລີ໌ຍ-ຫລັດ

(ເປີ໊ຍ ໝັ່ງ ດສ່ຳ ເລີ໌ຍ-ຫລັດ 5:1-21 )

1 ທິນ-ຮູ່ງ ກ໊ອງ ນ້າຍ ເຕີ໋ຍ ຢຽດ ຕສູ໊ງ ຫວ່າ. 2 “ເຢຍ ເຊ ຕສ໊ຽວ, ເມີ່ຍ ບົວ ເຍີຍ ທິນ-ຮູ່ງ, ຕົ້ງ ຕໍ້ ເມີ່ຍ ບົວ ທສວດ ອີ^ຢິ໊ບ ເຕີ່ຍ-ປູງ, ຕສຸ ໂຕສ໋ວ ໂນ່ວ ເຍີຍ ເຕີ່ຍ-ປູງ ວົ້ວ ເຕົ້າ. 3 “ທສວດ ລິ໌ວ ເຢຍ, ໄມ້ ຕູ໋ງ ຕສັ່ງ ກັ້ນ ເຕົ້າ ຕສ້ຽນ. * ທສວດ ອີ^ຢິ໊ບ 20:23 ; ດສ່ຳ ເລີ໌ຍ-ຫລັດ 5:7 4 “ໄມ້ ຕູ໋ງ ເຫວີ່ຍ ກັ້ນ ໂຕສ໋ວ ຫາຍ ຫຍູ່ງ ໝິ່ວ-ຝັງ. ໄມ້ ກຸ໊ນ ຢຽມ ກູ້ ຫງວ່າຍ ລູ່ງ, ຢຽມ ກະເດ໊ຍ ເດົາ, ໄຟ ຢຽມ ເດົາ-ເດ໊ຍ ເຍີຍ ວວມ ກະຍົ໌ວ ຫາຍ ຫຍູ່ງ ເຍີຍ ຝັງ. * ທສວດ ອີ^ຢິ໊ບ 20:23 ; 34:17 ; ເລ^ວີ 19:4 ; 26:1 ; ດສ່ຳ ເລີ໌ຍ-ຫລັດ 4:15-19 ; 5:8 ; 27:15 5 ໄມ້ ຕູ໋ງ ເກວີ່ຍ ເອຈ່ຍ ປ໋າຍ ນ້າຍ ເຕີ໋ຍ ໝິ່ວ-ຝັງ ຢ່າ ໄມ້ ຕູ໋ງ ຕສັ່ງ, ເຫວີ່ຍ ຕສຸ ເຢຍ, ຕສ໊ຽວ, ເມີ່ຍ ບົວ ເຍີຍ ທິນ-ຮູ່ງ, ແດສງ ຈ໊ຽນ ເມີ່ຍ ບົວ. ເຢຍ ອ໋ອຍ ຕີ່ງ ດສອງ ເຢຍ ວົ້ວ ເຕີ໋ຍ ມ່ຽນ ເຍີຍ ຕສຸ໌ຍ, ຕີ່ງ ເຖົາ ນິ່ນ ບົວ ເຍີຍ ປົວ ເປຢີຍ ຕ່ອຍ ເຕສີ໊ຍ-ຟຸນ. * ທສວດ ອີ^ຢິ໊ບ 34:6-7 , 14; ຊ້າວ ມ່ຽນ 14:18 ;
ດສ່ຳ ເລີ໌ຍ-ຫລັດ 4:24 ; 5:9-10 ; 7:9-10 6 ມປ່າ ຫາຍ ເຕົ້າ ຮນ້ຳ ເຢຍ, ເອີຍ ຈ໊ຽນ ເຢຍ ເຍີຍ ເລີ໌ຍ-ຫລັດ ໂຕສ໋ວ, ເຢຍ ຕສ່ຽວ ຮນ້ຳ ນິ່ນ ເຈ໋ຍ ມີ່ງ ເຖົາ ຢຽດ ທສິນ ຕ່ອຍ. * ຮິບ^ລູ ຫວ່າ ຂູງ ກ໊ອງ ເຖົາ ທສິນໆ. ມ່າຍ ເຕີ໋ຍ ໂຊວ ຟານ ຕ້າຍ ກ໊ອງ ເຖົາ ຢຽດ ທສິນ ຕ່ອຍ ໄຟ ເຖົາ ທສິນໆ ຕ່ອຍ ໄຟ ເຖົາ ທສິນໆ ເຕົ້າ ມ່ຽນ. 7 “ໄມ້ ຕູ໋ງ ຫລະໄກວ້ ຫລົ່ງ ຕສ໊ຽວ, ເມີ່ຍ ບົວ ເຍີຍ ທິນ-ຮູ່ງ, ເຍີຍ ບົ໋ວ. ຫາຍ ເຕົ້າ ຫລະໄກວ້ ຫລົ່ງ ຕສ໊ຽວ ເຍີຍ ບົ໋ວ, ຕສ໊ຽວ ອ໋ອຍ ຕີ່ງ ວົ້ວ ເຕົ້າ ເຍີຍ ຕສຸ໌ຍ. * ເລ^ວີ 19:12 ; ດສ່ຳ ເລີ໌ຍ-ຫລັດ 5:11
8 “ອ໋ອຍ ຕສຸ ຈັ໋ງ ຈ໊ຽນ ຫລົ່ງ ຕີ້ງ ກົງ ຮນອຍ ເປ໋ນ ນອມ ທສີງ-ເດສ່ງ ຮນອຍ. * ທສວດ ອີ^ຢິ໊ບ 16:23-30 ; 23:12 ; 31:12-17 ; 34:21 ; 35:2-3 ; ເລ^ວີ 19:3 ; ດສ່ຳ ເລີ໌ຍ-ຫລັດ 5:12-15
9 ມ່າຍ ຈຸ໊ ຮນອຍ ໂຕສ໋ວ ເມີ່ຍ ບົວ ເຍີຍ ຢຽດ ຕສູ໊ງ ກົງ. 10 ມປ່າ ຕະທສຽດ ຮນອຍ ວົ້ວ ຮນອຍ ເຊ ຕີ້ງ ກົງ ຮນອຍ ເຫວີ່ຍ ຝູງ ປຸນ ຕສ໊ຽວ. * ທສວດ ອີ^ຢິ໊ບ 23:12 ; 31:15 ; 34:21 ; 35:2 ; ເລ^ວີ 23:3 ວົ້ວ ຮນອຍ ໄມ້ ຕູ໋ງ ໂຕສ໋ວ ຫາຍ ຫຍູ່ງ ກົງ. ເມີ່ຍ ບົວ ກັ້ນ ເຖສົາ ເມີ່ຍ ບົວ ເຍີຍ ນ່າມ ໂຍ໌ວ, ເມີ່ຍ ບົວ ເຍີຍ ໂນ່ວ ເກົ໊າ ໂນ່ວ ເປີ໌ຍ, ເມີ່ຍ ບົວ ເຍີຍ ແຊງ-ຄູ້, ໄຟ ຢຽມ ເມີ່ຍ ບົວ ເຍີຍ ຕສີ້ງ ເຍີຍ ກັ້ນ ຝີງ ມ່ຽນ ຢ່າ ໄມ້ ຕູ໋ງ ໂຕສ໋ວ ຫາຍ ຫຍູ່ງ ກົງ. 11 ເຫວີ່ຍ ຕສຸ ຕສ໊ຽວ ຫລົ່ງ ຈຸ໊ ຮນອຍ ເຕສີ໋ຍ ລູ່ງ ເຕສີ໋ຍ ເດົາ ເຕສີ໋ຍ ຄ້ອຍ ເຖສົາ ຢຽມ ນ້າຍ ເຕີ໋ຍ ກະຍົ໌ວ ເຍີຍ ຢຽດ ຕສູ໊ງ ກະນ້າຍ. ມປ່າ ເສ່ຍ ຮນອຍ ວົ້ວ ຮນອຍ ທິນ-ຮູ່ງ ຕີ້ງ ກົງ ອ໊ະ. ເຫວີ່ຍ ນ້າຍ ຕສ໊ຽວ ເຖສີຍ ຟຸ ວົ້ວ ນອມ ຮນອຍ ເປ໋ນ ນິ່ນ ເຍີຍ ທສີງ-ເດສ່ງ ຮນອຍ. * ທິນ ເຕີ່ຍ ໂຕ້ວ 2:1-3 ; ທສວດ ອີ^ຢິ໊ບ 31:17 12 “ອ໋ອຍ ຕສຸ ຖົງ ໜິ່ມ ເມີ່ຍ ບົວ ເຍີຍ ເຕ໋ຍ ໝ່າ, ເມີ່ຍ ບົວ ຖສີງ ຕ້າຍ ຕສຢັ້ງ ຕຸ໊ ສ່ຽວ ຍ່ວນ ດ໊າວ ຢຽມ ຕສ໊ຽວ, ເມີ່ຍ ບົວ ເຍີຍ ທິນ-ຮູ່ງ, ຈິວ ປຸນ ເມີ່ຍ ບົວ ເຍີຍ ເຕີ່ຍ-ປູງ. * ເລ^ວີ 19:3 ; ດສ່ຳ ເລີ໌ຍ-ຫລັດ 5:16 ; 27:16 ; ມັດ^ທາຍ 15:4 ; 19:19 ;
ມາ^ໂຄ 7:10 ; 10:19 ; ລູ^ກາ 18:20 ; ເອ^ເຟ^ໂຊ 6:2-3 13 “ໄມ້ ຕູ໋ງ ໄຕ໋ ມ່ຽນ. * ທິນ ເຕີ່ຍ ໂຕ້ວ 9:6 ; ເລ^ວີ 24:17 ;
ມັດ^ທາຍ 5:21 ; 19:18 ; ມາ^ໂຄ 10:19 ; ລູ^ກາ 18:20 ; ໂລ^ມາ 13:9 ; ຢາ^ກອບ 2:11 14 “ໄມ້ ຕູ໋ງ ຫຮຢຽນ ເອົ໊າ ຫຮຢຽນ ໂອກ໋. * ເລ^ວີ 20:10 ; ດສ່ຳ ເລີ໌ຍ-ຫລັດ 5:18 ; ມັດ^ທາຍ 5:27 ; 19:18 ; ມາ^ໂຄ 10:19 ; ລູ^ກາ 18:20 ;
ໂລ^ມາ 13:9 ; ຢາ^ກອບ 2:11 15 “ໄມ້ ຕູ໋ງ ໂຕສ໋ວ ຕສະ. * ເລ^ວີ 19:11 ; ດສ່ຳ ເລີ໌ຍ-ຫລັດ 5:19 ;
ມັດ^ທາຍ 19:18 ; ມາ^ໂຄ 10:19 ; ລູ^ກາ 18:20 ; ໂລ^ມາ 13:9 16 “ໄມ້ ຕູ໋ງ ໂຕສ໋ວ ຈ໊າ ຕສອງ-ເຕສ໋ງ ຫ່ອຍ ມ່ຽນ. * ທສວດ ອີ^ຢິ໊ບ 23:1 ;
ດສ່ຳ ເລີ໌ຍ-ຫລັດ 5:20 ; ມັດ^ທາຍ 19:18 ; ມາ^ໂຄ 10:19 ; ລູ^ກາ 18:20
17 “ໄມ້ ຕູ໋ງ ເມົ້າ ມ່ຽນ ເຍີຍ ເປຢົ໊າ. ໄມ້ ຕູ໋ງ ເໝວີ່ຍ ເຈັ໊ຍ ມ່ຽນ ເຍີຍ ເອົ໊າ, ມ່ຽນ ເຍີຍ ໂນ່ວ ເກົ໊າ ໂນ່ວ ເປີ໌ຍ, ມ່ຽນ ເຍີຍ ງົ່ງ ໄຟ ລໍ່ ໄຟ ນິ່ນ ມ່າຍ ເຍີຍ ຫາຍ ຫຍູ່ງ ກະນ້າຍ.” * ໂລ^ມາ 7:7 ; 13:9

ອິ^ສະ^ລາ^ເອນ ມ່ຽນ ກ້ຳ ເດສ໋ຍ ໄຫ່

(ເປີ໊ຍ ໝັ່ງ ດສ່ຳ ເລີ໌ຍ-ຫລັດ 5:22-23 )
